# I'm Pierre-Louis 👋
I'm a Computer Science student based in France.

My passion for IT never stops growing since the beginning of my studies. I discovered this world and was already amazed by it when I was a child.

What really makes me love developing is the fact that I can I build anything I want and helps me understand how a computer works.
Moreover, as I am someone who loves more than ever learning things, software engineering being a extreme vast universe, makes me love it even more.

My favorite technologies are **C** (for system projects) and **TypeScript** (for web projects). I am still discovering frameworks since I am a beginner going to an advanced level, but for now, I really appreciate **NestJS** for backend development and **Angular** or also the **React** library for frontend one.

My main domain is currently the **full-stack** development, but I would truely appreciate to understand the world of **embedded systems**.

# My projects 📚
I have made a bunch of projects for school such as [SuperHero Manager](https://github.com/Plouis-Dnx/miniprojet) which deeply learned me how to create a clean full-stack project and introced me to very useful technologies that are the frameworks.
[StreamSa](https://github.com/Plouis-Dnx/StreamSa) also educated me in the same way how to write a clean code and introduced me the *SOLID* principles that I now use and think about instinctively when building a project.

But my **best project** still is, to date, [QuickTalk](https://github.com/Plouis-Dnx/QuickTalk), an instant messaging app developed as part of my Bachelor's project. It shows that I have tried to apply the methods learned from the lessons but also that I have understood that using technologies (frameworks, websockets, ...) are the very foundation of a true advanced project.  
Over that, it reinforced even more my passion and made me want to develop other big projects.
