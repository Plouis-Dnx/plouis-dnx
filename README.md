# I'm Pierre-Louis 👋
I'm a Computer Science student based in France.

My passion for IT has never stopped growing since I started my studies. I discovered this world when I was a child and was already amazed by it.

What really makes me love development is the fact that I can build anything I want and that it helps me understand how a computer works.
Moreover, since I love learning new things more than anything, software engineering being a extremely vast universe, makes me love it even more.

My favorite technologies are **C** (for system projects) and **TypeScript** (for web projects). I am still discovering frameworks as I move from beginner to advanced, but for now, I really appreciate **NestJS** for backend development and **Angular** or also the **React** library for frontend work.

My main domain is currently the **full-stack** development, but I would truly appreciate to understand the world of **embedded systems**.

# My projects 📚
I have built several school projects such as [SuperHero Manager](https://github.com/Plouis-Dnx/miniprojet) which taught me how to create a clean full-stack project and introduced me to very useful technologies that are the frameworks.
[StreamSa](https://github.com/Plouis-Dnx/StreamSa) also helped me, in the same way,  how to write a clean code and introduced me the *SOLID* principles that I now use and think about instinctively when building a project.

But my **best project** so far is [QuickTalk](https://github.com/Plouis-Dnx/QuickTalk), an instant messaging app developed as part of my Bachelor's project. It shows that I have tried to apply the methods learned from the lessons but also that I have understood that using technologies (frameworks, websockets, ...) are the very foundation of a true advanced project.  
Beyond that, it reinforced even more my passion and made me want to develop other big projects.
